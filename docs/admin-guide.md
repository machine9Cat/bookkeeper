## 用户管理

在 **管理 → 用户管理** 页面可管理所有用户。

### 用户列表

表格显示所有用户的以下信息：
- 用户名
- 密码（明文，方便管理员查阅）
- 角色（admin / user）
- 状态（pending / approved / disabled）
- 注册时间

### 操作

| 操作 | 说明 |
|------|------|
| ✅ 通过 | 审核通过待注册用户，允许其登录 |
| 🔑 重置密码 | 将用户密码重置为新密码 |
| 🚫 禁用 | 禁止用户登录（可用于离职人员） |
| ✅ 启用 | 恢复被禁用的账户 |
| ❌ 删除 | 删除用户（仅当该用户无录入记录时可用） |

### 注意事项

- 管理员不能禁用自己的账户
- 管理员不能删除自己
- 已禁用用户登录时会提示"账户已被禁用，请联系管理员"

## 审核流程

审核是管理员对支出记录进行确认的机制。

### 为什么需要审核？

- 确保数据的准确性和合规性
- 已审核的记录不可修改或删除（包括管理员）
- 需要修改已审核记录时，必须先取消审核

### 审核操作

**单条审核：**
1. 在明细页面找到目标记录
2. 点击该行的 **✅ 审核** 按钮
3. 记录状态变为"已审核"

**取消审核：**
1. 点击已审核记录的 **↩️ 取消审核** 按钮
2. 记录恢复为可编辑状态

**批量审核：**
1. 勾选多条未审核记录
2. 点击 **批量审核** 按钮
3. 所有选中记录一次性审核通过

### 审核状态

记录在表格中的"审核"列显示状态：
- 绿色 **已审核** 标签 — 已审核通过
- 灰色 **待审核** 标签 — 等待审核

## 字典管理

在 **管理** 页面可管理以下字典数据：

| 字典 | 说明 | 系统预设 |
|------|------|----------|
| 📦 大类 | 物品分类 | 16 个预设分类 |
| 💳 付款方式 | 支付方式 | 现金/银行卡/支付宝/微信/信用卡 |
| 🏪 购买渠道 | 采购来源 | 线下/网购/批发/代购 |
| 👤 经手人 | 经办人员 | 无预设 |
| 📝 物品名称 | 物品字典 | 用于自动补全 |
| 🔧 规格型号 | 规格字典 | 用于自动补全 |
| ⚖️ 单位 | 计量单位 | 25 个预设单位 |

### 操作

- **添加** — 输入名称后点击"添加"按钮
- **删除** — 点击条目右侧的"删除"按钮（系统预设项不可删除）

添加的字典项不会影响已有记录，仅用于后续录入时的选项和自动补全。

## 备份管理

### 本地备份

在 **备份** 页面的"本地备份"卡片：

| 操作 | 说明 |
|------|------|
| 📥 下载数据库 | 直接下载当前数据库文件 |
| 📤 上传恢复 | 上传 `.db` 文件覆盖当前数据库（自动备份原库为 `.bak`） |
| 历史备份 | 显示所有 `.bak` 文件，可下载或删除 |

### 自动备份

系统会在数据变更 **30 分钟** 后自动创建本地备份，文件名为 `bookkeeper.db.auto.{时间戳}`。

### 云备份（WebDAV）

支持坚果云、NextCloud 等 WebDAV 服务。

**配置示例（坚果云）：**
1. 登录坚果云，进入"安全设置"
2. 生成**应用密码**
3. 在备份页面填写：
   - WebDAV 地址：`https://dav.jianguoyun.com/dav/`
   - 用户名：坚果云账号邮箱
   - 密码：应用密码（非登录密码）
   - 远程路径：`/bookkeeper/`

**操作：**
- **保存配置** — 保存备份设置
- **立即备份** — 立即上传压缩备份到远程
- **恢复数据** — 从远程下载最新备份并恢复

> 备份文件使用 gzip 压缩后上传，可缩小 80-90%。

## 数据导入导出

### Excel 导入

支持 `.xlsx` 和 `.xlsb` 格式。导入过程分两个阶段：
1. **计数阶段** — 分析文件行数
2. **处理阶段** — 逐行导入并显示进度

导入完成后会显示成功/失败统计。

### 报表导出

在统计页面选择月份后：
- **导出 Excel** — 月度明细报表
- **导出 PDF** — 月度汇总报表

> 导出时按钮显示"导出中..."，防止重复点击。
