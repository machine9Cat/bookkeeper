## 登录与注册

### 注册

1. 首次访问时自动进入登录页
2. 点击"注册账号"链接
3. 填写用户名和密码
4. 第一个注册的用户自动成为**管理员**
5. 后续注册的用户需等待管理员审核通过后方可登录

### 登录

输入用户名和密码登录。登录失败时会给出具体提示：
- **用户名不存在** — 账号未注册
- **密码错误** — 密码不正确

## 概览

概览页显示当前月份的整体情况：

- **月度总支出** — 当月所有支出的合计金额
- **记录总数** — 当月记账条数
- **大类统计** — 按物品大类分组的支出金额
- **经手人统计** — 按经手人分组的支出金额
- **月度趋势图** — 近 12 个月支出趋势折线图
- **分类饼图** — 各分类占比
- **各维度排行** — 大类/付款方式/经手人/渠道支出排行

> 概览数据按 **订单日期（expense_date）** 统计。

## 记账

记账页面用于录入新的支出记录。

### 表单字段

| 字段 | 说明 |
|------|------|
| 物品大类 | 下拉选择，可选系统预设或管理员添加的分类 |
| 物品名称 | 文本输入，支持自动补全已有物品名称 |
| 规格型号 | 文本输入，支持自动补全已有规格 |
| 数量 | 输入数量（整数） |
| 单位 | 下拉选择（个/只/套/件/箱/包/瓶/盒/条/双/台/张/把/根/米/千克/斤/两/升/毫升/罐/卷/袋/副/块） |
| 单价 | 输入单价，与数量自动计算总价 |
| 总价 | 输入总价，或由数量×单价自动计算 |
| 订单日期 | 支出发生的日期 |
| 经手人 | 下拉选择，支持自动补全 |
| 付款方式 | 下拉选择 |
| 购买渠道 | 下拉选择 |
| 发票 | 发票号码（可选） |
| 发票类型 | 发票类型（可选） |
| 备注 | 备注信息（可选） |

### 保存规则

- **入库** 自动设为"是"
- **入账** 自动设为"否"
- **登记日期** 自动填入当前日期（不手动显示）

## 明细查询

明细页面以表格形式展示所有支出记录。

### 筛选条件

- **大类** — 按物品大类过滤
- **经手人** — 按经手人过滤
- **付款方式** — 按付款方式过滤
- **购买渠道** — 按购买渠道过滤
- **订单日期范围** — 按开始/截止日期过滤
- **搜索** — 按物品名称或规格全文搜索

### 表格操作

- **排序** — 点击表头可按该列排序
- **行内编辑** — 点击表格中的数据单元格可直接修改（已审核的记录不可编辑）
- **分页** — 每页 20 条记录
- **列显隐** — 点击"⚙️ 列"下拉菜单可切换列的显示/隐藏（设置自动保存）

### 快速新增

点击筛选栏右侧的 **➕ 新增** 按钮展开内联表单，可在不离开明细页面的情况下快速添加记录。

### 批量操作

勾选多条记录后可使用：
- **批量审核**（管理员） — 一次审核多条记录
- **批量删除** — 一次删除多条未审核记录

### Excel 导入

点击 **📥 导入 Excel** 按钮上传 Excel 文件进行批量导入。导入进度会实时显示。

## 统计报表

统计页面按月汇总支出数据。

### 查看统计

1. 选择要查看的月份
2. 点击 **查询** 按钮
3. 查看各维度的汇总数据

### 导出报表

- **📥 导出 Excel** — 下载为 `.xlsx` 格式
- **📄 导出 PDF** — 下载为 PDF 格式

> 统计与报表均按 **订单日期（expense_date）** 统计，不与登记日期混合。

---

## 修改密码

1. 点击导航栏 **🔑 修改密码**
2. 输入原密码和新密码
3. 点击确认保存

如忘记密码，请联系管理员重置。
